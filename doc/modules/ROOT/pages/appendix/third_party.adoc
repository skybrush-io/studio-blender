= Third party tools
:imagesdir: ../../assets/images

The following third party tools might be of help throughout your drone show design process with **Skybrush Studio for Blender**.

If you know of any other tool you think should be on this list, please mailto:support@collmot.com[let us know].

== Blosm for Blender

Blosm for Blender, formerly known as Blender-OSM lets you import Google 3D cities, OpenStreetMap 3D buildings and global real-world terrain into your Blender project with a few clicks.

* https://prochitecture.gumroad.com/l/blender-osm[Free version]
* https://prochitecture.gumroad.com/l/blosm[Pro version]

== Murmur

Murmur is a swarm flocking simulation for Blender and **Skybrush Studio**, based on the https://www.science.org/doi/full/10.1126/scirobotics.aat3536[Vásárhelyi et al. 2018 optimized flocking model].

The code for the simulation is prepared by our Discord user @envyofthebirds, representing the company https://illuminaero.com/[Illuminaero].

* https://github.com/illuminaero/murmur[Download open-source code]

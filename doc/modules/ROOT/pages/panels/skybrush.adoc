= Skybrush tab
:imagesdir: ../../assets/images
:experimental:

The Skybrush tab in the sidebar of the 3D Viewport editor is the first tab you need to use when starting to design a show in *Skybrush Studio for Blender*. This tab contains general configuration options for your show design workflow.

image::panels/skybrush.jpg[Skybrush tab]

The Skybrush tab contains four panels:

* xref:panels/skybrush/setup.adoc[Setup panel]
* xref:panels/skybrush/show.adoc[Show panel]
* xref:panels/skybrush/swarm.adoc[Swarm panel]
* xref:panels/skybrush/drone_groups.adoc[Drone groups]
